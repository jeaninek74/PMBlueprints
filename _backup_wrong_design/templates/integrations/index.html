Platform Integrations

Connect PMBlueprints with your favorite project management tools

M

Monday.com

Work OS for project management

Export templates directly to Monday.com boards with automatic column mapping and task creation.

Setup Instructions:

  1. Go to Monday.com → Profile → Admin → API
  2. Generate a new API token
  3. Copy the token and paste it in Settings
  4. Test the connection
Configure {% if settings.monday_api_token %} {% endif %}
S

Smartsheet

Collaborative work management

Create Smartsheet sheets from templates with pre-configured columns, formulas, and automation.

Setup Instructions:

  1. Go to Smartsheet → Account → Apps & Integrations
  2. Click "API Access" and generate token
  3. Copy the token and paste it in Settings
  4. Test the connection
Configure {% if settings.smartsheet_api_token %} {% endif %}
G

Google Sheets

Cloud-based spreadsheets

Export templates to Google Sheets with automatic formatting, formulas, and sharing options.

Setup Instructions:

  1. Go to Google Cloud Console
  2. Enable Google Sheets API
  3. Create OAuth 2.0 credentials
  4. Authorize PMBlueprints access
Configure {% if settings.google_sheets_credentials %} {% endif %}
M

Microsoft 365

OneDrive & SharePoint

Save templates directly to OneDrive or SharePoint with automatic versioning and collaboration features.

Setup Instructions:

  1. Go to Azure Portal
  2. Register PMBlueprints application
  3. Grant Microsoft Graph permissions
  4. Authorize access
Configure {% if settings.microsoft_access_token %} {% endif %}
